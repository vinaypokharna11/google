GOOGLE LENS

Search what you see

Google Lens in Chrome lets you search for things you can’t describe in words, track down that cool sweater a character on a show is wearing, ask for pros and cons when you’re deciding between items, and even translate road signs in photos.

Learn more about Lens in Chrome

TAB ORGANIZER

Smartly organize your tabs

With Tab Organizer, Chrome will automatically suggest and create groups based on your open tabs. Multitaskers, this one’s for you —it’s particularly useful if you’re working on several projects in Chrome at the same time, like planning a trip, researching a topic, and shopping.

Learn more about Tab organizer

GENERATIVE THEMES

Express yourself with generative themes

Quickly generate custom themes based on a subject, mood, visual style and color that you choose — no need to become an AI prompt expert! Imagine scenes like an “aurora borealis” that’s “illustrated” with a “serene” mood, and see what Chrome comes up with.

Learn more about AI themes

GEMINI SHORTCUT

Get more creative with Gemini

It's easy to start a chat with Gemini right from the desktop address bar. You can get help with creative and complex tasks, like planning a trip, learning more about a new topic, or writing a job-winning cover letter.

Learn more about Gemini shortcut

HELP ME WRITE

Write with confidence

Writing on the web doesn’t have to be daunting. We’ve made it easier to put your thoughts into words, whether you want to leave a review, craft a friendly RSVP, or make an inquiry about an apartment rental.
